Detailed Description

The ProfileDB class.

Create an sqlite3 table in a database file. The database structure: Column 1: ProfileID INTEGER PRIMARY KEY Column 2: FullName TEXT Column 3: Photo TEXT Column 4: Description TEXT Column 5: ScrapbookID INTEGER UNIQUE
