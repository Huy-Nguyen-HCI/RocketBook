Detailed Description

The BlogDB class.

Create an sqlite3 table named "Blogs" in a database file. The database structure: Column 1: BlogID INTEGER PRIMARY KEY Column 2: AccountID INTEGER NOT NULL Column 3: ScrapbookID INTEGER NOT NULL Column 4: BlogTitle TEXT Column 5: BlogContent TEXT Column 6: Privacy INTEGER NOT NULL
