Detailed Description

The TweetDB class.

Create an sqlite3 table named "Tweets" in a database file. The database structure: Column 1: TweetID INTEGER PRIMARY KEY Column 2: AccountID INTEGER NOT NULL Column 3: ScrapbookID INTEGER NOT NULL Column 4: TweetContent TEXT Column 5: Privacy INTEGER NOT NULL
