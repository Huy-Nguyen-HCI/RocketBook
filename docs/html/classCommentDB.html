Detailed Description

The CommentDB class.

Create an sqlite3 table named "Comments" in a database file. The database structure: Column 1: CommentID INTEGER PRIMARY KEY Column 2: AccountID INTEGER NOT NULL Column 3: BlogID TEXT Column 4: CommentContent TEXT NOT NULL
