Detailed Description

The MultimediaDB class.

Create an sqlite3 table named "Multimedias" in a database file. The database structure: Column 1: MultimediaID INTEGER PRIMARY KEY Column 2: AccountID INTEGR NOT NULL Column 3: ScrapbookID INTEGER NOT NULL Column 4: MultimediaTitle TEXT Column 5: MultimediaDescription TEXT Column 6: MultimediaContent TEXT Column 7: Privacy INTEGER NOT NULL
