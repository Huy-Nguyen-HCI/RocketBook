Detailed Description

The AccountDB class.

Create an sqlite3 table named "Accounts" in a database file. The database structure: Column 1: AccountID INTEGER PRIMARY KEY Column 2: Username TEXT UNIQUE Column 3: Password TEXT Column 4: ProfileID INTEGER Column 5: AdminRights
